# Welcome to the Student Guide

Artemis is an interactive learning platform that supports programming exercises, quizzes, modeling tasks, text exercises, and exams with automatic and manual assessment. This guide covers most information you need as a student.

## Getting Started

A typical student workflow follows these steps:

1. **[Enroll in a course](/student/getting-started/courses)** and explore available content
2. **[Work on exercises](/student/exercises/intro)** including programming, quiz, modeling, and text exercises
3. **[Attend lectures](/student/learning-content/lectures)** and use the AI tutor for help
4. **[Review feedback](/student/assessment-grades/assessment)** on your submissions and check your grades
5. **[Track your progress](/student/progress-analytics/learning-analytics)** using learning analytics and adaptive learning paths

<Callout variant={CalloutVariant.info}>
**New to Artemis?** Start with [Getting Started](/student/getting-started) to learn how to enroll and navigate the platform, then head to [Exercises](/student/exercises/intro) to begin working on assignments.
</Callout>
