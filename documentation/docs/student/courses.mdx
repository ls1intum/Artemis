# Customizing Courses

Courses contain exercises, exams and lectures.
Roles include **Users**, **Tutors**, **Editors**, **Instructors**, each with different permissions.

---

## Student-Facing Behaviour

Students can register for a course if **Student Course Registration Enabled** is active.
Registration happens via:

<Image src={btnCourseRegister} alt={"course register"} size={ImageSize.small} style={{display:"inline",verticalAlign:"middle"}} />
