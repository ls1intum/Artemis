# Welcome to the Instructor Guide

Artemis is an interactive learning platform that supports programming exercises, quizzes, modeling tasks, text exercises, file uploads, and exams with automatic and manual assessment. This guide covers everything you need to run a course.

## Getting Started

A typical instructor workflow follows these steps:

1. **[Request a course](/instructor/course-management/course-requests)** through the built-in course request form, then configure its settings once an administrator has approved it
2. **Add content** such as exercises, lectures, and exams
3. **Set up communication** through channels, FAQs, and tutorial groups
4. **Assess and grade** by reviewing submissions, defining grading keys, and checking for plagiarism
5. **Monitor progress** using learning analytics and adaptive learning paths

<Callout variant={CalloutVariant.info}>
**New to Artemis?** Start with [Course Management](/instructor/course-management) to learn how to request and configure your course, then head to [Exercises](/instructor/exercises/intro) to create assignments for your students.
</Callout>
