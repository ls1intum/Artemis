# File Upload Exercise

## Overview

Conducting a file upload exercise consists of 3 steps:

1. Setup of the exercise in Artemis
2. Submission by a student
3. Assessment by the staff

## Setup

The following sections describe the supported features and the process of creating a new file upload exercise.

- Open <img src={courseManagement} alt="Course Management" style={{display: 'inline', verticalAlign: 'middle', height: '2rem'}} />.
- Navigate into **Exercises** of your preferred course.

<Image src={courseManagementDashboard} alt="Course Management Dashboard" size={ImageSize.large} />

- Click on **Create a new file upload exercise**.

<Image src={createNewFileUploadExercise} alt="Create New File Upload Exercise" size={ImageSize.large} />

- Fill out all mandatory values and click on <img src={saveButton} alt="Save Button" style={{display: 'inline', verticalAlign: 'middle', height: '2rem'}} />.
- The exercise-specific **File Pattern** defines which file types students can upload as solution. The input field accepts all file endings without leading dot separated by commas.

<Image src={fileUploadExerciseCreationPattern} alt="File Upload Exercise Creation Pattern" size={ImageSize.large} />

- Result: **File Upload Exercise**.

<Image src={courseDashboardExerciseFileUpload} alt="Course Dashboard Exercise File Upload" size={ImageSize.large} />

- Click the <img src={editButton} alt="Edit Button" style={{display: 'inline', verticalAlign: 'middle', height: '2rem'}} /> button of the file upload exercise to update the configuration and assessment instructions.
- You can get an overview of the exercise by clicking on the title.

## Student Submission

- When the exercise is released, the students can upload the requested file.
- They can choose a file with <img src={chooseFileButton} alt="Browse Button" style={{display: 'inline', verticalAlign: 'middle', height: '2rem'}} /> and then submit the file with <img src={submitButton} alt="Submit Button" style={{display: 'inline', verticalAlign: 'middle', height: '2rem'}} />.

<Image src={fileUploadExerciseStudentView} alt="File Upload Exercise Student View" size={ImageSize.large} />

- After a file is uploaded it can then be downloaded again by clicking on **Download file**.

<Image src={fileUploadExerciseStudentViewSubmitted} alt="File Upload Exercise Student View Submitted" size={ImageSize.large} />

## Assessment

- When the due date is over, you can assess the submissions. From the assessment dashboard, go to exercise the assessment dashboard of the file upload exercise.
- There you can assess the submitted student submissions, by first downloading the file, and then adding feedback with points.

<Image src={fileUploadExerciseAssessment} alt="File Upload Exercise Assessment" size={ImageSize.large} />
